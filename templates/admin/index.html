STARTUP GUIDE

IMPORTANT REMINDERS

  • Never delete all data from admin table.

  • Make sure to give people_id to all members and make sure it is unique for each member.

  • All images uploaded will be stored in static/uploads folder with the filename it already had.

  • If you wish to update any images. First delete its equivalent data from the table and then upload the new image

  • You can update 6 papers per request in Update Papers. Adding more would still yield result, but they will not have journal or citation details.

Admin Table

Add name and password here if you want to have a diffrent login credentials.
Never delete all data from this table.
Always make sure atleast one record exist.

Home Page

Add text to show in the home screen here.
You can also add pictures to show in the slides here.
View/Delete them in the TABLE Home Page Image.
Add images using the function Upload Home page Images

People Page

You can add or remove members to the group here
There are 9 tables and 2 functions in this menu.
Upload Image can be used to upload image of the member.
Update Papers can be used to update publication details.

Mandatory

  • People (TABLE) :- Add basic data of members here.
    Make sure to give people_id to all and make sure it is unique.

Optional

  • PeopleImage (TABLE) :- View/Delete added images of members here.

  • Contacts (TABLE) :- Add contact details of members here.
    Make sure to give the corresponding people_id of the member.
    You can add both numbers and email ids here

  • Academics (TABLE) :- Add academic details of members here.
    Make sure to give the corresponding people_id of the member.

  • Research (TABLE) :- Add research interst of members here.
    Make sure to give the corresponding people_id of the member.

  • Awards (TABLE) :- Add award details of members like scholarships won, here.
    Make sure to give the corresponding people_id of the member.

  • Positions (TABLE) :- Add details of positions hold by member.
    Make sure to give the corresponding people_id of the member.

  • Papers (TABLE) :- Add publication details of members here.
    Make sure to give the corresponding people_id of the member.

  • MTech (TABLE) :- Update MTech Alumni members here.

  • Upload Image (FUNCTION) :- Add a picture of the member here.
    Make sure to give the corresponding people_id of the member.

  • Update Papers (FUNCTION) :- Update publication here. This is automated.
    If the journal and citation details are missing.
    Try re-running the function with the same limits
